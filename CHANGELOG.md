# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-10-18

### Added
- Initial release of masstemplate CLI tool
- Basic template application functionality
- Support for local template directories in `~/.local/masstemplate/`
- Install scripts (pre/post install hooks)
- Processing DSL for advanced file manipulation
- Copier.yml support for variable templating
- Command-line interface with subcommands: list, apply, info
- Modular CLI architecture with separate command modules
